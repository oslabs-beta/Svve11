Button

  • Source: https://github.com/Svve11/
  • WAI-ARIA: https://w3c.github.io/aria-practices/#button

A button is a widget that enables users to trigger an action or event, such as submitting a form, opening a dialog, canceling an action, or performing a delete operation.

Installation

Installation

Import the component in the script section of your Svelte file:


              import ❴ Button ❵ from 'svve11'
Usage

Usage

Creating a Button

A button instance can be created by placing the code below in the body of your Svelte file. 


              <Button {id} {label} {content} {handleClick} {style}/>

Each attribute should be defined as follows:

  • id (string): sets the id attribute of the button component.
  • label (string): sets the aria-label attribute of the button component.
  • content (string): sets the text that is displayed inside the button component.
  • handleClick (function): defined the action or event to be triggered when the button is clicked.
  • style (string): This attribute is optional and sets the styles of the button component.

Example Button Code:


              <Button 
                id="demo-button-1" 
                label="accessible-button-1" 
                content="Click me!" 
                handleClick={() => console.log('You clicked a button!')} 
                style="height: 50px; width:300px;"
              />
